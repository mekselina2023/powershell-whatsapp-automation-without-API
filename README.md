Employee WhatsApp Automation
API-free WhatsApp automation solution built with PowerShell and Excel.
Overview
This project automates employee communication through WhatsApp Web without requiring any paid messaging API.
The solution reads employee information from an Excel file, validates data, matches personalized documents, and automatically sends messages and files to each recipient.
Features
✅ No paid API required
✅ No WhatsApp Business API subscription
✅ Excel-based employee management
✅ Automatic phone number validation
✅ Employee name verification
✅ Personalized file delivery
✅ Batch processing
✅ Activity logging
✅ PowerShell automation
Workflow
Reads employee records from Excel.
Validates employee names and phone numbers.
Checks whether the required document exists.
Matches each employee with their corresponding file.
Sends personalized WhatsApp messages.
Attaches employee-specific documents.
Records success and error logs.
