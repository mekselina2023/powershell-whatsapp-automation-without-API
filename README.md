# Employee WhatsApp Automation

API-free WhatsApp automation solution built with PowerShell and Excel.

## Overview

This project automates employee communication through WhatsApp Web without requiring paid messaging APIs.

## Features

- No paid API required
- No WhatsApp Business subscription
- Excel-based employee management
- Automatic phone number validation
- Employee name verification
- Personalized file delivery
- Batch processing
- Activity logging

## Workflow

1. Read employee records from Excel
2. Validate employee data
3. Locate employee-specific files
4. Send personalized WhatsApp messages
5. Attach documents
6. Save delivery logs

- WhatsApp Web
